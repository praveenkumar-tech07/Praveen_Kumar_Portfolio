## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/praveenkumar-tech07/Praveen_Kumar_Portfolio.git
cd Praveen_Kumar_Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env.local` file and add your Formspree ID:
```
NEXT_PUBLIC_FORMSPREE_ID=your_formspree_id
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
